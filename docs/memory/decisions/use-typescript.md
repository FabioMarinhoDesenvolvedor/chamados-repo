# Use TypeScript (Full-Stack)

Data: 2026-06-25

## Contexto
Projeto fullstack com NPM. Precisa de tipagem consistente entre front e back para evitar bugs em runtime, especialmente na definição de chamados, prioridades e roles de usuário.

## Decisão
TypeScript em todo o stack (frontend + backend).

## Consequências
- Tipos compartilhados entre front e back
- Refatoração mais segura
- Curva inicial de setup ligeiramente maior
- Build step obrigatório
