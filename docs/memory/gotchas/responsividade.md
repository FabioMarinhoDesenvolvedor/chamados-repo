# Responsividade Obrigatória

Problema:
Todos os componentes DEVEM funcionar em mobile e desktop. Qualquer feature nova que não seja responsiva é considerada incompleta.

Solução:
- Desenvolver mobile-first (breakpoints de baixo pra cima)
- Testar em viewport 375px (iPhone SE) como mínimo
- Sidebar colapsa em drawer no mobile
- Tabelas viram cards empilhados no mobile
- Touch targets mínimos de 44x44px
